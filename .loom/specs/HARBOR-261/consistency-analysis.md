# Consistency Analysis

- Harbor remains the owner of browser process, CDP and session facts.
- Challenge redirects are visible public facts, not task success or authentication proof.
- Deep title failure degrades only to page-list metadata; version failure is not hidden.
- Abort handling reuses existing CdpClient cleanup and adds no dependency or retry abstraction.
- App, Core, Lode, sensitive material, and external write behavior are unchanged.
